打开后应该是一个一无所有的界面，如下图：(部分不该有的东西我用马赛克挡住了)

点击左上角的加号，选择SFTP，给自己的服务器取个名(我写的是hello)，然后在下图中填入以下信息：

  • Host: 填写你服务器的ip
  • User name: 登陆用户名
  • Authentication: 选择登陆方式(可以选择密码登陆，也可以选择免密码登陆)
  • Root path: 服务器上工程的根目录
