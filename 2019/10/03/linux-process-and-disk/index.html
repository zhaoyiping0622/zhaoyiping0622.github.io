pidof命令

查看某个进程的进程id

例如：pidof init

ps命令

ps命令主要是静态的查看进程，也就是只有在ps命令运行时那一刻的进程信息会显示出来。下面给出手册上提供的几个例子，其他更多参数可以运行man ps查看。

显示系统里所有进程(标准格式):

  • ps -e
  • ps -ef
  • ps -eF
  • ps -ely

显示系统里所有进程(BSD格式):

  • ps ax
  • ps axu

打印进程树：

  • ps -ejH
  • ps axjf

得到线程信息：

  • ps -eLf
  • ps axms

得到安全信息：

  • ps -eo euser,fuser,f,comm,label
  • ps axZ
  • ps -eM

显示所有root运行的进程：

  • ps -U root -u root u

显示PID为42的进程名字：

  • ps -q 42 -o comm=

top命令

top命令实时的给出系统各项信息。

第一行给出了系统运行时间，登陆用户数量、1分钟/5分钟/15分钟负载情况。

第二行是当前进程数量，总共4个(total)，1个运行(running)，3个休眠(sleeping)，0个stopped，0个僵尸态(zombie)。

第三行是CPU使用情况。

第四行是内存使用情况。

第五行是交换空间使用情况。

第七行及以下是各个进程的情况。

第七行从左往右依次是：进程号、进程所有者、进程优先级、nice值、使用的虚拟内存总量、使用的未被换出的物理内存大小、共享内存大小、进程状态、上次更新到现在的CPU时间占用百分比、进程使用的物理内存百分比、进程使用的CPU时间总计、进程名称

几个使用技巧

b:高亮当前运行进程

x:打开/关闭排序列的高亮效果

<</>>:改变排序关键字

pgrep命令

使用类似于grep的匹配模式查找进程。

终止进程

kill命令

kill [options] […]

通过ps或top命令查到对应进程pid以终止进程。

pkill命令

pkill [options] pattern

使用类似于grep的匹配模式终止进程。

磁盘管理

Linux所有设备都以文件形式存在于/dev目录下。

设备文件分两种：块设备文件和字符设备文件

常见设备文件：

  • /dev/hd[a-t]: IDE设备
  • /dev/sd[a-z]: SCSI设备
  • /dev/fd[0-7]: 标准软驱
  • /dev/md[0-31]: 软raid设备
  • /dev/loop[0-7]: 本地回环设备
  • /dev/ram[0-15]: 内存
  • /dev/null: 无限数据接收设备
  • /dev/zero: 无限零资源
  • /dev/tty[0-63]: 虚拟终端
  • /dev/ttyS[0-3]: 串口
  • /dev/lp[0-3]: 并口
  • /dev/console: 控制台
  • /dev/fb[0-31]: framebuffer
  • /dev/cdrom -> /dev/hdc
  • /dev/modem -> /dev/ttyS[0-9]
  • /dev/pilot -> /dev/ttyS[0-9]
  • /dev/random: 随机数设备
  • /dev/urandom: 随机数设备

Linux 磁盘管理常用的三个命令df,du,fdisk。

df

df - report file system disk space usage

显示文件系统的磁盘使用量。

df [OPTION] … [FILE] …

参数：

  • -a,—all: 显示所有文件系统
  • -B,—block-size=SIZE: 设置块大小
  • -h —human-readable: 大小设为1024进制
  • -H —si: 大小设为1000进制
  • -i,—inodes: 以inodes数量来显示
  • -T,—print-type: 显示文件系统类型

du

du - estimate file space usage

查看文件/目录对磁盘空间的使用。

du [OPTION]… [FILE]…
du [OPTION]… —files0-from=F

参数：

  • -0,—null: 每行以NUL结束
  • -a,—all: 显示所有文件，而不只是目录
  • —apparent-size: 显示文件实际大小而不是占用空间
  • -B,—block-size=SIZE: 设置块大小
  • -b,—bytes: 类似于’—apparent-size —block-size=1’
  • -d,—max-depth=N: 设置显示层数(展开几层目录)
  • -h,—human-readable: 以人类可读形式给出
  • —inodes: 以inodes数量来显示
  • -k: —block-size=1K
  • -l,—count-links: 一个硬链接大小计算多次
  • -m: —block-size=1K
  • -S,—separate-dirs: 不计算子目录
  • —time: 显示最后修改时间
  • -X,—exclude-from=FILE: 排除的文件

fdisk

fdisk - manipulate disk partition table

fdisk是Linux磁盘分区表操作工具。(极其危险，使用时需要格外小心谨慎)

fdisk [-l] device
fdisk [options] [device…]

参数：

  • -l: 显示所有分区。

指定设备文件后即可对磁盘进行分区操作。

mkfs

mkfs - build a Linux filesystem

对磁盘进行文件系统格式化。

mkfs [-t 文件系统格式] 装置文件名

mount

mount - mount a filesystem

磁盘挂载和删除。

mount [-l|-h|-V]
mount -a [-fFnrsvw] [-t fstype] [-O oplist]
mount [-fnrsvw] [-o options] device|dir
mount [-fnrsvw] [-t fstype] [-o options] device dir
