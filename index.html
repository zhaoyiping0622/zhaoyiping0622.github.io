sudo apt install tightvncserver

输入vncserver进行第一次启动，需要输入一个6到8位的密码。vncserver会在用户目录下生成一个.vnc目录，需要将里面的xstartup修改为如下：

#exec /etc/X11/xinit/xinitrc
